# Lesson 01 – Fundamentals of AI & Machine Learning

## Overview
Artificial Intelligence (AI) is a broad field focused on building systems that can perform tasks requiring human intelligence such as learning, reasoning, and decision-making.

Machine Learning (ML), Deep Learning (DL), and Generative AI are subsets of AI that enable systems to learn from data and generate intelligent outputs.

---

## Key Concepts

### Artificial Intelligence (AI)
- Umbrella term for intelligent systems
- Includes ML, DL, and Generative AI

### Machine Learning (ML)
- Systems learn from data to improve performance
- No explicit programming required

### Deep Learning (DL)
- Uses neural networks (brain-inspired)
- Handles complex data like images, audio, video

### Generative AI
- Generates new content (text, images, etc.)
- Built on foundation models (FMs)

---

## Machine Learning Fundamentals

### 1. Data Types
- **Labeled Data** → Has input + output (used in supervised learning)
- **Unlabeled Data** → Only input (used in unsupervised learning)

#### Structured Data:
- Tabular data (CSV, databases)
- Time-series data

#### Unstructured Data:
- Text, images, audio, video

---

### 2. Learning Types

- **Supervised Learning** → Learn from labeled data  
- **Unsupervised Learning** → Find patterns in unlabeled data  
- **Reinforcement Learning** → Learn via rewards/penalties  

---

### 3. ML Lifecycle

1. Data Collection  
2. Data Preparation  
3. Model Training  
4. Evaluation  
5. Deployment  

 *“Garbage in = Garbage out” → Data quality is critical*

---

### 4. Inferencing

- **Batch Inferencing** → Process large data at once  
- **Real-time Inferencing** → Instant predictions (chatbots, etc.)

---

## Deep Learning Basics

- Based on **Neural Networks**
- Layers:
  - Input Layer  
  - Hidden Layers  
  - Output Layer  

### Applications:
- Computer Vision (image recognition)
- NLP (chatbots, translation)

---

## Generative AI Fundamentals

### Foundation Models (FMs)
- Pretrained on massive datasets
- Can perform multiple tasks:
  - Text generation  
  - Image generation  
  - Q&A  

---

### FM Lifecycle

1. Data Selection  
2. Pre-training  
3. Optimization  
4. Evaluation  
5. Deployment  
6. Continuous Improvement  

---

### LLM Concepts

- **Tokens** → Smallest text units  
- **Embeddings** → Numerical meaning of words  
- **Vectors** → Represent relationships  

---

### Other Models

- **Diffusion Models** → Generate images from noise  
- **Multimodal Models** → Combine text + image + audio  

---

### Optimization Techniques

- Prompt Engineering  
- Fine-tuning  
- RAG (Retrieval-Augmented Generation)
